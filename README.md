# Classe do Herói e Ataque

Projeto desenvolvido como parte do desafio de lógica de programação da DIO, com foco na utilização de classes em JavaScript.

---

## Descrição

O objetivo do projeto é representar um herói de uma aventura utilizando uma classe, contendo propriedades como nome, idade e tipo.

Além disso, foi implementado um método responsável por simular um ataque, exibindo uma mensagem de acordo com o tipo do herói.

---

## Lógica do Projeto

Cada tipo de herói possui um tipo de ataque específico:

| Tipo       | Ataque            |
|------------|-------------------|
| Mago       | magia             |
| Guerreiro  | espada            |
| Monge      | artes marciais    |
| Ninja      | shuriken          |

O método `atacar()` utiliza essa lógica para retornar a mensagem correta.
